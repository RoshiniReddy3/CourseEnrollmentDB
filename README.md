
# Course Enrollment Database (SQL)

A simple SQL project that models a course enrollment system with students, instructors, courses and enrollments.

## Files

- `schema.sql` – Creates the database and tables  
- `insert_data.sql` – Inserts sample data  
- `queries.sql` – Example SELECT, JOIN, GROUP BY and HAVING queries  

This project is meant to demonstrate basic SQL skills including database design, constraints, sample data handling and query writing.
